# LaTeX-Cord
Powercord plugin to send LaTeX in the current channel.

## Installation
`cd powercord/src/Powercord/plugins && git clone https://github.com/amitpalbadhan/LaTeX-Cord`

In the directory where Powercord is installed.

## Usage
Format:
`.latex {...}`

## Example
`.latex x^2 + y^2 = z^2`
